## Getting Started

Ensure Flutter and Git is installed.

1. Install dependencies:

```sh
flutter pub get
```

2. Configure git hooks:

```sh
git config core.hooksPath .githooks/
```

3. Generate routes:

```sh
flutter pub run build_runner build --delete-conflicting-outputs
```
